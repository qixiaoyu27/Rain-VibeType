# 雨音输入法 1.0.0 使用说明

这份说明面向第一次体验 Rain 的 Windows 11 用户。

## 安装

1. 打开 [Rain Vibetype Releases](https://github.com/qixiaoyu27/Rain-VibeType/releases/latest)。
2. 下载文件名以 `_x64-setup.exe` 结尾的安装器。
3. 双击安装器并按提示完成安装。
4. 如果 Windows SmartScreen 显示“已保护你的电脑”，请确认文件来自 `qixiaoyu27/Rain-VibeType` 官方 Release，再点击“更多信息” → “仍要运行”。当前体验版没有商业代码签名证书，因此可能出现此提示。

## 首次配置

1. 启动雨音输入法。
2. 在首次配置窗口点击“下载并自动配置”。
3. Rain 会根据 SenseVoice Small 的映射自动下载轻量原生推理框架，然后下载并校验模型。
4. 等待首页显示“可以开始使用”。下载期间请保持网络连接，不要退出应用。

模型和推理组件体积较大，建议至少预留 10 GB 磁盘空间。它们只会在你明确点击后下载；语音识别仍在本机完成。

## 开始语音输入

1. 打开微信、浏览器、记事本、编辑器或其他需要输入文字的应用。
2. 点击目标输入框，把光标放到准备写字的位置。
3. 按住 `Ctrl + Shift + Space` 开始说话。
4. 说完后松开快捷键，等待识别结果自动写入。

录音或识别期间按 `Esc`，可以取消本次输入。

## 常用设置

- 修改快捷键：打开“录音与快捷键”，点击当前组合键，再按新的组合键并保存。
- 切换录音方式：可选择“按住说话，松开识别”或“按一次开始，再按一次结束”。
- 测试麦克风：在“录音与快捷键”中选择输入设备并执行 0.5 秒测试。
- 修改写入方式：在“文字写入”中选择“剪贴板粘贴（推荐）”或“模拟逐字输入”。
- 开启文本整理：在“文本整理”中点击“下载并自动配置”，Rain 会同时安装 Qwen 模型和它所需的 llama.cpp 推理框架，再开启开关。
- 关闭开机启动：在“应用设置”中关闭“登录 Windows 时启动”。

## 托盘与退出

关闭主窗口后，Rain 默认继续驻留在 Windows 系统托盘，快捷键仍然可用。需要彻底退出时，请在托盘图标菜单中选择“退出”。

## 常见问题

### 按快捷键没有反应

- 在“录音与快捷键”里检查快捷键是否与其他软件冲突。
- 尝试重新录制一个包含非修饰键的组合键。
- 确认 Rain 仍在系统托盘中运行。

### 提示尚未配置模型或推理组件

打开“模型管理”，下载并选择 SenseVoice Small。Rain 会自动补齐它所需的推理框架；网络中断后可以继续下载，不必从头开始。

### 识别结果没有写到原输入框

为了避免误写，Rain 会在识别完成时再次确认目标应用。如果你在录音后切换到了其他应用，结果只会复制到剪贴板；回到目标位置后手动粘贴即可。

### GPU 模式加载失败

按提示回退到 CPU，或在“推理设置”中把推理设备改为 CPU。CPU 功能完整，只是大型模型可能更慢。

### 如何卸载

先从托盘彻底退出 Rain，再到 Windows“设置 → 应用 → 已安装的应用”中卸载“雨音输入法”。模型和推理组件位于应用数据目录；从应用内删除模型时，最后一个引用它的模型被删除后，对应推理框架也会自动清理。

## 隐私

Rain 不保存录音或识别历史，音频和识别文字在本机处理。默认安全粘贴会临时使用剪贴板，并在不覆盖用户新复制内容的前提下恢复原剪贴板。
