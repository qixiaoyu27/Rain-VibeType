- [ ] 安装目录和应用数据目录不存在录音文件

- [ ] 存在任何已安装引用时拒绝卸载推理框架，删除最后一个引用模型后自动删除框架
- [ ] 模型删除无法越过应用管理的模型根目录
- [ ] 原生 SenseVoice、Python CPU / NVIDIA Worker 分别在无 Python 的干净 Windows 11 x64 虚拟机启动
- [ ] 基础 NSIS 不包含 PyTorch/CUDA、llama.cpp 或任何模型权重，首次安装后按需下载组件
- [ ] NSIS 安装、升级、卸载和开机启动行为正确
- [ ] GitHub Releases 更新清单与产物签名验证成功
- [ ] 用户拒绝更新时不会下载或安装

## 性能记录

正式验收时分别记录，不把模型加载/推理时间混入应用处理时间：

- 快捷键触发到开始录音：目标不超过 100 ms。
- 浮条出现：目标不超过 150 ms。
- 已加载模型时停止录音到发送音频：目标不超过 100 ms。
- Worker 返回到开始注入：目标不超过 100 ms。
- 空闲常驻 CPU 接近 0%，未加载模型时不占用 GPU 显存。
